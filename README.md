# Bluegreen

## Usage

### Help

```bash
bluegreen -h
Usage of ./bluegreen:
  -bluegreen string
        blue or green? Default: blue (default "blue")
```

### Green

```bash
bluegreen -bluegreen green
```

```bash
curl localhost:8888
```

returns:

```bash
green
```

### Blue

```bash
bluegreen
```

```bash
curl localhost:8888
```

returns:

```bash
blue
```
